# prep-javascript-variables-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- What does the `=` (equals) operator do in JavaScript?
  The assignment operator assigns a value to a variable.

- What operator will return the data type of a variable value?
  The typeof operator will return the data type of a variable value
- Which keyword is used to declare a variable in JavaScript?
  The keyword used to declare a variable in JavaScript is let (or var).
- Which characters is a JavaScript variable allowed to begin with?
  You can also begin a variable name with the $ character.
- What are quotation marks used for in JavaScript?
  Quotation marks are used to indicate a string
- What is the purpose of strings in JavaScript?
  The purpose of strings are to output text in JavaScript
- What is the purpose of booleans in JavaScript?
  The purpose of booleans are to see if an expression or variable have the values of true or false. Boolean types only have 2 values: true or false.
- What is the purpose of numbers in JavaScript?
  The purpsoe of numbers in JavaScript is to represent floating point values, like numbers that we use in math. Ex: 295, 37.8, etc.
- What does `null` mean in JavaScript?
  Null in JavaScript means unknown or empty value.
